Installation

Clone the repo:

git clone https://github.com/masala-DosAi/webchatbot.git
cd webchatbot

Install dependencies:

pip install -r requirements.txt

Set up API key in .env:

GOOGLE_API_KEY=your_api_key_here

Run the App

flask run

Visit http://127.0.0.1:5000/ in your browser.
